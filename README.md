

**AI Desktop Voice Assistant Using Python**

An AI-powered desktop voice assistant built entirely in Python that brings the experience of a personal intelligent assistant — like Siri, Google Assistant, or Amazon Alexa — directly to your computer. The assistant, named **James**, operates through natural spoken language, listening to what you say, understanding your intent, and responding in clear, human-like speech. No touch, no typing — just your voice.

The core of the system is built on Google's Speech Recognition API, which captures audio from your microphone in real time and converts it accurately into text, even with Indian English accents and varied speech patterns. Responses are delivered back through text-to-speech synthesis, making every interaction feel natural and conversational. When no specific command is matched, the assistant seamlessly shifts into full AI chat mode powered by OpenAI's GPT, allowing open-ended conversations, answering complex questions, and maintaining context across the entire session — just like talking to a real intelligent assistant.

Beyond conversation, James is a genuinely useful productivity tool. Ask it to search Google and it will open the results and simultaneously read out a Wikipedia summary so you get an answer immediately without lifting a finger. Ask about a movie and it pulls the title, release year, full plot, IMDb rating, runtime, director, and top cast members and reads them all aloud. Ask for the news and it fetches live top headlines on any topic you name, reading them out one by one. Need to know the weather in a city? Just ask. Want to send an email? Dictate the content and James sends it through Gmail without you touching the keyboard.

The assistant handles everyday desktop tasks just as smoothly. It can open YouTube, Wikipedia, Google, Notepad, and VS Code by voice. It plays your local music library, opens your webcam, captures and saves screenshots with a filename you dictate, translates any text into any language on the spot, sets timed reminders that fire in the background while the assistant keeps listening, and evaluates math expressions spoken aloud. There is even a built-in number guessing game for when you want a break.

Under the hood, the project is thoughtfully engineered. Reminders run on separate background threads so they never interrupt the main listening loop. API integrations — News API, OpenWeatherMap, IMDb, Google Translate — are cleanly separated and handled with proper error recovery so the assistant never crashes on a bad response. All AI-generated responses from the GPT mode are automatically saved to disk as text files for future reference. Sensitive credentials like API keys and email passwords are kept entirely out of the codebase through a configuration system, making the project safe to share and open source.

This project was developed as a major academic submission at **B.P. Poddar Institute of Management & Technology, Salt Lake Campus, Kolkata**, exploring real-world applications of artificial intelligence, natural language processing, speech recognition, and Python software development. It demonstrates how modern AI tools and APIs can be combined into a single cohesive system that genuinely improves how a person interacts with their computer — making computing more accessible, more hands-free, and more intelligent.
